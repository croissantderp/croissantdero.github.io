Geckoinator

Geckoinator

A mosaic maker which replaces pixels in an image with smaller images that closest matches the pixel's color.

After the basic framework was finished, I introduced a feature to improve the quality of the images using Floyd-Steinberg dithering. I also introduced batch processing and multithreading in order to be able to process videos.
